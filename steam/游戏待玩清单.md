# 🎮 游戏待玩清单

> [!tip] 说明
> 本清单根据推荐数和游玩时长自动分级，推荐数越高口碑越好，优先级越高。

## 📊 库存概览

```dataview
TABLE length(rows) AS 数量
FROM "steam/Steam Games"
GROUP BY choice(playtime = 0, "🎮 从未游玩", choice(playtime < 2, "🕐 仓促体验", choice(playtime < 10, "🔄 初步体验", choice(playtime < 50, "⚔️ 深度游玩", "🏆 资深玩家"))))
```

## 🏆 必玩神作（推荐数 > 500000 且游玩 < 10 小时）

口碑炸裂但你还没深入体验，强烈建议优先开坑：

> [!cards]
> ```dataview
> TABLE WITHOUT ID
>   "![](" + cover + ")" AS 封面,
>   file.link AS 游戏,
>   round(playtime, 1) + " h" AS 游玩时长,
>   recommendations AS 推荐数,
>   tags AS 类型
> FROM "steam/Steam Games"
> WHERE playtime < 10 AND recommendations > 500000
> SORT recommendations DESC
> ```

## ⭐ 热门佳作（推荐数 100000~500000 且游玩 < 10 小时）

广受好评的高质量游戏，值得投入时间：

> [!cards]
> ```dataview
> TABLE WITHOUT ID
>   "![](" + cover + ")" AS 封面,
>   file.link AS 游戏,
>   round(playtime, 1) + " h" AS 游玩时长,
>   recommendations AS 推荐数,
>   tags AS 类型
> FROM "steam/Steam Games"
> WHERE playtime < 10 AND recommendations >= 100000 AND recommendations <= 500000
> SORT recommendations DESC
> ```

## 👍 口碑好游（推荐数 50000~100000 且游玩 < 10 小时）

评价不错的游戏，有空可以一试：

> [!cards]
> ```dataview
> TABLE WITHOUT ID
>   "![](" + cover + ")" AS 封面,
>   file.link AS 游戏,
>   round(playtime, 1) + " h" AS 游玩时长,
>   recommendations AS 推荐数,
>   tags AS 类型
> FROM "steam/Steam Games"
> WHERE playtime < 10 AND recommendations >= 50000 AND recommendations < 100000
> SORT recommendations DESC
> ```

## 🎲 小众精品（推荐数 < 50000 且游玩 < 2 小时）

可能被忽略的宝藏游戏，适合换口味时尝试：

> [!cards]
> ```dataview
> TABLE WITHOUT ID
>   "![](" + cover + ")" AS 封面,
>   file.link AS 游戏,
>   round(playtime, 1) + " h" AS 游玩时长,
>   recommendations AS 推荐数,
>   tags AS 类型
> FROM "steam/Steam Games"
> WHERE playtime < 2 AND recommendations < 50000 AND recommendations > 0
> SORT recommendations DESC
> ```

## 📋 全部待玩一览（游玩 < 2 小时）

所有几乎没玩过的游戏，按推荐数从高到低排列：

> [!cards]
> ```dataview
> TABLE WITHOUT ID
>   "![](" + cover + ")" AS 封面,
>   file.link AS 游戏,
>   round(playtime, 1) + " h" AS 游玩时长,
>   recommendations AS 推荐数,
>   tags AS 类型
> FROM "steam/Steam Games"
> WHERE playtime < 2
> SORT recommendations DESC
> ```

## 🏷️ 待玩游戏 · 按类型分布

从未游玩和仓促体验的游戏，按类型统计数量：

```dataview
TABLE length(rows) AS 待玩数量
FROM "steam/Steam Games"
WHERE playtime < 2
FLATTEN tags AS genre
GROUP BY genre
SORT length(rows) DESC
```
